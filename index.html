BeaverTail.me

Previous Next

BeaverTail is a crowd sourced user-submitted news platform that places news on an easily navigated world map. Users have the ability to verify, refute and add context to each news article submission. They can also use the integrated forum for further discussions or to bring attention to any issues.

This web application was built using Node.js with Express and EJS, along with MySQL database. The forum integration was implemented via phpBB.

LastBytes

LastBytes is a waste scanner that leverages camera object identification and machine learning to identify waste and recommend an appropriate waste bin (for example, paper or compost). Through the app's recommendations, the aim is to reduce incorrectly sorted waste at the SAP Vancouver office. This app was created as a part of D-Shop, which is SAP's community innovation space.

This android mobile application is implemented with Flutter as the UI framework for its compatibility with Android and iOS devices.

Boulder Buddy is social media platform for all types of rock climbers to connect and meet up. Currently, the main functionality allows users to connect with one another via chat in real-time.
